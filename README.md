# DevilBlues
Devil Blues é um jogo de ação onde o jogador controla um guitarrista que fez um pacto com o diabo para alcançar fama e talento. Após o diabo quebrar o acordo e roubar sua alma antes do tempo, deixando ela presa no tártaro , o músico inicia uma jornada para conseguir fugir e a alma dele ser livre. Durante essa jornada, ele enfrenta diversos inimigos que o impedem de sair.
Jogo 3D em primeira pessoa
